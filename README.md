# Custom Document Chatbot

An AI-powered chatbot that answers questions using uploaded documents and AI knowledge.

## Features

- **Document Processing**: Upload PDF, DOCX, TXT, MD files
- **Smart Answering**: Uses document content when available, AI knowledge when needed
- **Semantic Search**: Finds relevant content in documents
- **Modern UI**: Clean, responsive interface with source tracking
- **Fast & Efficient**: Quick document processing and answering
